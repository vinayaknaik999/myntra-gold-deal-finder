# Privacy Policy

This Chrome extension does not collect, transmit, or sell any personal data.

## Data stored
The extension stores user-provided settings (24K rate, 22K rate, refresh interval) using Chrome's storage API.

## Data not collected
- No browsing history
- No personally identifiable information
- No payments or account information
- No analytics/tracking

## Network access
The extension does not send any data to external servers.
